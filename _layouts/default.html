Exception Rules

stack

Stacked icons are a special format from font awesome that allows for layering one icon on another to make a more complex icon. This comes with some trade offs between ease of use and what can be achieved. This is an exception to the norm and should be used sparingly and is often used with more complex icons used for products. stacked documentation

illustration

Illustration are used on icons that have a more flourishes look when in a large format or print. The Image reference is used whenever the specific icon is going to be used larger then 150px. This is an exception to the norm and should be used sparingly and is often used with more complex icons used for products.
